# **App Name**: DDI Replicated

## Core Features:

- Dashboard Replication: Faithfully recreate the dashboard layout, with all interactive elements, widgets and panels, from the original design.
- Visual Fidelity: Precisely replicate all visual elements of the original DDI web design.
- Responsive Recreation: Mimic all responsive behaviors, including layout adjustments and breakpoints from the original design.
- Interactive Element Reproduction: Replicate animations, transitions, and micro-interactions as present in the original design.
- Visual Fine-tuning: Match every detail of the original design's typography. Accurately replicate colors.

## Style Guidelines:

- Primary color: Dark lime green (#3DD579) for brand identity and emphasis.
- Background color: Dark charcoal (#121212) to provide contrast and focus.
- Accent color: Deep olive green (#91BA5E) to complement the primary.
- Font pairing: 'Inter' (sans-serif) for both headline and body text.
- Replicate custom icons consistent with the DDI aesthetic. Follow sizes, colors, and styles from the provided designs for navigational and functional elements.
- Accurately reproduce grid and flexbox layout, including sizes, margins, padding, alignment, and responsive behavior. Match layout structure pixel-perfectly.
- Replicate transitions and dynamic UI elements using smooth, subtle effects. Follow easing curves, animation durations, and state-dependent animations, as in the original.
